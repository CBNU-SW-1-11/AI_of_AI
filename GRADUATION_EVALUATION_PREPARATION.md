# 🎓 졸업작품 평가 준비 가이드

## 📋 시스템 개요

### 핵심 기능
1. **다중 AI 모델 응답 수집**: GPT-4o-mini, Gemini, Claude, HyperCLOVA 등 여러 모델의 답변을 동시에 수집
2. **심판 모델(Judge) 검증**: GPT-4o-mini를 심판 모델로 사용하여 각 응답을 분석하고 최적 답변 생성
3. **사실 검증 소스 검색**: Wikipedia, Wikidata, DBpedia, DuckDuckGo를 활용한 외부 검증
4. **영상 분석 및 프레임 검색**: YOLO, DeepFace, Ollama(llava), BLIP을 활용한 영상 분석
5. **채팅 히스토리 관리**: 모델 변경 시 자동으로 대화 초기화 또는 유지

---

## ❓ 교수님들이 물어볼 만한 질문 및 답변

### 1. 이 시스템의 핵심 차별점은 무엇인가요?

**답변:**
- 단일 AI 모델이 아닌 **다중 AI 모델의 응답을 수집**하여 비교 분석
- **심판 모델(Judge)**을 통해 각 응답의 신뢰도를 평가하고 최적 답변 생성
- **외부 검증 소스**(Wikipedia, Wikidata 등)를 활용한 사실 검증
- **영상 분석 기능**: 객체 감지, 얼굴 인식, 캡션 생성, 키워드 기반 프레임 검색

**데모 질문:**
- "대한민국 11대 대통령은?" → 검증 소스와 함께 답변 제공
- "마스코트가 등장하는 장면 보여줘" → 관련 프레임 이미지 반환

---

### 2. 검증 소스는 어떻게 작동하나요?

**답변:**
1. **질문 유형 분류**: LLM이 질문을 "사실적(factual)" 또는 "의견(opinion)"으로 분류
2. **키워드 추출**: 사실적 질문의 경우, LLM이 검증에 사용할 핵심 키워드를 추출
3. **병렬 검색**: Wikipedia, Wikidata, DBpedia, DuckDuckGo를 동시에 검색
4. **일치도 분석**: 검색 결과와 각 AI 응답의 일치도를 분석하여 신뢰도 평가
5. **검증 소스 제공**: 사용자에게 검증 소스 정보를 함께 제공

**데모 질문:**
- "팀 버너스리는 누구인가요?" → Wikipedia 검증 소스 제공
- "월드와이드웹 WWW 창시자는?" → 검증 소스와 함께 답변

---

### 3. 여러 AI 모델의 답변이 다를 때 어떻게 처리하나요?

**답변:**
1. **심판 모델 분석**: 각 AI 응답을 심판 모델이 분석하여 신뢰도 평가
2. **검증 소스 비교**: 검증 소스와의 일치도를 확인
3. **상호모순 감지**: 응답들 간의 모순을 감지
4. **프리미엄 모델 보팅**: 상호모순이 감지되면 프리미엄 모델들의 추가 응답을 수집하여 보팅
5. **최적 답변 생성**: 검증 소스와 가장 일치하는 답변을 선택하거나 통합

**데모 질문:**
- "대한민국 10대 대통령은?" → 여러 모델의 답변을 비교하고 검증 소스로 확인

---

### 4. 영상 분석은 어떻게 이루어지나요?

**답변:**
1. **객체 감지**: YOLO를 사용하여 프레임별 객체 감지
2. **얼굴 인식**: DeepFace를 사용하여 인물의 나이, 성별, 감정 분석
3. **캡션 생성**: Ollama(llava)와 BLIP을 사용하여 각 프레임의 자연어 캡션 생성
4. **프레임 검색**: 사용자 질문에 따라 키워드, 색상, 인물 등으로 관련 프레임 검색
5. **의도 기반 검색**: 색상 검색, 인물 검색, 시간대 검색, 요약 등 의도에 맞는 프레임 반환

**데모 질문:**
- "초록색 옷 입은 사람 찾아줘" → 관련 프레임 이미지 반환
- "마스코트가 등장하는 장면 보여줘" → 마스코트 관련 프레임 반환
- "어린이가 나오는 장면 찾아줘" → 어린이 관련 프레임 반환

---

### 5. 실제로 검증이 잘 작동하나요?

**답변:**
- 네, 사실적 질문에 대해 Wikipedia 등에서 검증 소스를 찾아 신뢰도를 평가합니다.
- 검증 소스가 발견되면 사용자에게 "검증 소스: Wikipedia (신뢰도: 90%)" 형태로 제공합니다.
- 검증 소스와 일치하는 정보를 우선적으로 선택하여 최적 답변을 생성합니다.

**테스트 질문:**
- "대한민국 11대 대통령은?"
- "팀 버너스리는 누구인가요?"
- "월드와이드웹 WWW 창시자는?"

---

### 6. 채팅 히스토리는 어떻게 관리되나요?

**답변:**
- **모델 1-2개 변경**: 이전 대화 내용을 기억하여 맥락 유지
- **모든 모델 변경**: 대화를 초기화하여 새로운 대화 시작
- **세션 기반 관리**: 사용자별로 독립적인 대화 세션 유지
- Django 캐시를 활용한 효율적인 히스토리 관리

---

### 7. 영상에서 프레임 검색이 정확한가요?

**답변:**
- 네, 키워드 기반 검색과 색상 기반 검색이 구현되어 있습니다.
- 캡션에서 키워드를 매칭하여 관련 프레임을 찾습니다.
- 마스코트, 캐릭터, 색상, 인물 등 다양한 키워드로 검색 가능합니다.

- "마스코트가 등장하는 장면 보여줘" (Video ID 76)
