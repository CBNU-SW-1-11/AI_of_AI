# 답변 품질 분석 및 개선 방안

## 📊 현재 답변 품질 평가

### ✅ 잘 작동하는 부분

1. **GPT-4o-Mini**: 
   - ✅ 간결하고 정확한 답변
   - ✅ 영상 내용을 잘 이해함
   - ✅ 기술적 정보(프레임 수 등) 제거 잘 됨

2. **Claude-3.5-Haiku**:
   - ✅ 상세하고 구조화된 답변
   - ✅ 정확도 높음
   - ✅ 영상 컨텍스트 잘 이해

3. **통합 응답**:
   - ✅ 여러 모델의 답변을 잘 종합
   - ✅ 대체로 정확한 정보 제공

### ❌ 문제점

1. **Gemini-2.0-Flash-Lite**:
   - ❌ "실내인가요 실외인가요?" → "지금 있는 곳이 어디인지 알려주시면" (완전히 잘못된 답변)
   - ❌ "자동차가 나와?" → "응! 자동차가 나올 수 있지" (영상에 자동차 없는데 잘못된 답변)
   - ❌ 영상 컨텍스트를 무시하고 일반적인 답변 제공

2. **일관성 문제**:
   - 같은 질문에 대해 모델마다 다른 답변
   - 통합 응답이 잘못된 정보를 포함할 수 있음

## 🔧 개선 방안

### 1. 영상 컨텍스트 강화
프롬프트에 영상 정보를 더 명확하게 전달

### 2. 답변 검증 로직 추가
영상과 무관한 답변을 자동으로 필터링

### 3. Gemini 응답 품질 개선
Gemini 모델의 프롬프트를 더 구체적으로 수정

### 4. 통합 응답 시 잘못된 정보 제외
여러 모델의 답변 중 영상과 무관한 답변은 제외
